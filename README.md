# Vendor-performance-analysis-project
### Project Overview

Effective inventory and vendor management are crucial for maximizing profitability in the retail and wholesale industry. This project analyzes vendor, brand, sales, and inventory data to uncover performance gaps, cost inefficiencies, and profitability drivers.

By leveraging SQL for data extraction, Python for analysis, and Power BI for visualization, the project delivers actionable insights that help businesses improve pricing strategies, vendor selection, and inventory turnover.
